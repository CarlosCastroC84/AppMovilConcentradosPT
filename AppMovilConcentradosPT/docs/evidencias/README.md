# Evidencias

Esta carpeta se reserva para capturas, videos cortos y exportes PDF usados en la entrega del proyecto.

## Estructura sugerida

- `navegador/`
- `android-emulador/`
- `android-dispositivo/`
- `release/`
- `ios/`

## Regla de nombres

Usar el identificador de la matriz de evidencia:

`EVID-XX-modulo-ambiente-fecha.ext`

Ejemplos:

- `EVID-01-login-exitoso-android-2026-03-14.png`
- `EVID-06-carrito-persistente-android-2026-03-14.png`
- `EVID-20-aab-release-2026-03-14.png`

## Recomendaciones

- guardar la evidencia el mismo dia en que se ejecuta la prueba
- no mezclar archivos de otras aplicaciones
- si una prueba falla, guardar tambien la captura del fallo
